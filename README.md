# MOFA-forum-system

A web-based forum management system developed for the **Database Management Systems (DBMS)** course.

The system allows users to register, create posts, comment on discussions, and organize content using tags. It demonstrates database design, ER modeling, and relational database implementation using PHP and MySQL.

---

## Features

- User registration and login
- Create and manage posts
- Comment system
- Tag-based post organization
- Keyword search across posts and users
- Backend database management

---

## System Architecture

The system is implemented using:

- **PHP**
- **MySQL**
- **Apache (XAMPP environment)**
